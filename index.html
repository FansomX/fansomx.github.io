关于我 (About)

我是一名在德国慕尼黑工作的医学物理博士和AI研究员。我的研究核心是为在线自适应放射治疗 (Online Adaptive Radiotherapy, ART) 创造更快速、更精准的计算工具。

我主要致力于光子段剂量预测、射束眼睛视图(BEV)与患者解剖结构之间的插值，以及开发用于实时剂量计算的毫秒级内核。我坚信开源与可复现性是推动科学进步的关键。

项目 (Projects)

ConvLSTM BEV Dose Prediction

一个深度学习模型，利用卷积长短期记忆网络（ConvLSTM）从射束眼睛视图（BEV）预测放射治疗中的剂量分布。

  • PyTorch
  • CuPy
  • Deep Learning

Geant4 Segment Simulator

基于 Geant4 的蒙特卡洛模拟工具，用于精确计算放射治疗中单个光子段产生的剂量，为模型训练提供高精度数据。

  • C++
  • Geant4
  • ITK
  • Monte Carlo

Adaptive ART Pipeline

一个高性能计算管线，整合了快速剂量计算内核，旨在实现在线自适应放疗中的实时计划调整。

  • CUDA
  • C++
  • High-Performance Computing
